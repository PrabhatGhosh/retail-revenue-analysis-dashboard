\# Retail Revenue Analysis Dashboard



An interactive dashboard built using Python, Pandas, Plotly, and Streamlit for retail sales analysis.



\## Features



\- Revenue Analysis

\- Customer Insights

\- Product Insights

\- Geographic Insights

\- Time-Based Analysis

\- Business Recommendations



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Plotly

\- Streamlit



\## Run Locally



pip install -r requirements.txt



streamlit run "🏠 Overview.py"
